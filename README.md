# Curriculum-RUC
## Motivation
收集RUC各个课程的笔记, 复习资料, 大作业等等资源

## File Structure
- 每个学院对应一个文件夹, 其子文件夹为各个课程
- 一般来说每个文件夹下
  - `Review/`
    - 自己整理的复习资料
  - `Projects/`
    - 大作业, 实验, 实验报告
  - `Tasks/`
    - 小作业, 个人作业
