    \section*{11.1}
    设该进程所需的处理器运行时间为$T_p$，I/O设备的运行时间为$T_d$
    进程总相应时间为$T_e$
    无缓冲的情况下，$T_e = T_p + T_d$，加入单个缓冲区后，当$T_p = T_d$时，
    有$T_e = \frac{T_p+T_d}{2}$，这是所有情况中最短的时间。而最长时间的情况
    即$T_d$远远大(小)于$T_p$时。\par
    则$ \frac{T_p+T_d}{2} \leq T_e \leq T_p + T_d$
    得证。
